Project Overview

TâcheFlow is a full-stack web application that helps students organise and track their academic tasks with intelligent due-date awareness and real-time progress analytics.

🎯 Problem

Students struggle to keep track of deadlines across multiple courses

💡 Solution

A focused task manager with colour-coded urgency and one-click status toggling

🛠️ Stack

MERN — MongoDB, Express.js, React.js, Node.js

Security Implementation

🔒 Authentication

  • bcrypt password hashing (10 salt rounds)
  • JWT tokens with 7-day expiry
  • Token stored in localStorage
  • Auth middleware on all protected routes

🛡️ Data Protection

  • Users can only access their own tasks
  • Input validated by Mongoose schemas
  • CORS restricted to frontend origin
  • Secrets in .env (not in source code)
