Project Purpose

TâcheFlow was developed to address the challenge of academic task management for students. Many students struggle to keep track of deadlines, assignments, and project milestones across multiple courses. TâcheFlow provides a focused, distraction-free tool with intelligent due-date awareness and real-time progress analytics.

Introduction About the System

What is TâcheFlow?

TâcheFlow is a full-stack web application designed to help students manage their academic tasks efficiently. Built using the MERN stack (MongoDB, Express.js, React.js, Node.js), it provides a modern, responsive interface where students can create, organise, track, and complete their academic workload.

Problem Statement

University students often juggle multiple courses, each with their own assignments, projects, and deadlines. Without a dedicated tool, tasks are easily forgotten or deprioritised, leading to missed deadlines and poor academic performance. Existing tools are either too complex (Jira, Asana) or too simple (sticky notes, text files) for student needs.

Proposed Solution

TâcheFlow solves this by providing:

  • Quick task creation — Add a task with title, description, and due date in seconds
  • Visual urgency indicators — Colour-coded badges (Overdue, Today, Tomorrow, Soon, Normal) show what needs attention first
  • One-click status toggle — Switch tasks between pending and completed instantly
  • Live dashboard analytics — See total, pending, completed, and overdue counts at a glance
  • Filter system — View only the tasks you need (All / Pending / Completed / Overdue)
  • Secure accounts — Each student has their own private account with encrypted passwords

Database Schema:

User Collection:
├── name (String, required)
├── email (String, unique, indexed)
├── password (String, bcrypt hashed)
└── createdAt (Date)

Task Collection:
├── title (String, required)
├── description (String)
├── dueDate (Date)
├── status ("pending" | "completed")
├── userId (Reference → User)
└── createdAt (Date)
